## Subjects & Grades

- **Vocab** — Grades 4–5
- **Vocab (Hole Filling)** — Grades 9–10

## XP Rule

- 5 XP per activity
- 0 XP for repeated activities in the same word list
- Review tests (My Banked Words): 0.5 XP per correct word
- Full mastery of a word requires passing the proficiency test 3 times

## Accuracy Threshold

- 80% accuracy required for activities and review tests
